간단한 문장입력과 배치로 발표자료를 꾸며보세요.

대본 입력하러가기

EPT의 대본분석기가 입력받은 문장을 분석하여
발표자료에 들어갈만한 요소들을 만들어 줍니다.

더이상 발표자료의 디자인을
빈 화면으로부터 고민하지 마세요

당신의 발표 대본이나, 자료를 글로 입력하는 것으로 충분히
발표할 도구(pptx)를 준비할 수 있습니다.
자료가 빠르게 시각화되는 것을 경험해보세요!

대본이나 자료를 글로 입력해 넣고,
디자인은 "마치 쇼핑하듯이"
클릭하기만 하면 됩니다.

짧은 시간에, 더 많은 자료정리, 디자인 테스트를 손쉽게 할 수 있습니다.
이미 결정된 형태라도 손쉽게 다른 모양으로 바꿀 수 있다는 뜻이죠!

더이상 사진이나 동영상을 찾는 데
어려워하거나 시간을 많이 쓰며 고민하지 마세요!

Ept를 통해서 정리된 자료의 글들에 대해서
'구글', '유튜브' 등에 검색된 자료를 검토하실 수 있습니다.

또한, 저작권이 걱정되시는 많은 분들을 위해,
'픽사베이', '언스플래시'의 자료 역시 검토하실 수 있어요!

자료를 준비하실 때, 당연하게도, 손쉽게
사진/동영상의 배치나 내용을 테스트할 수 있다는 거죠!

Easy Presentation Tool의 사용법

Ept는 당신의 글(대본 또는 자료 정리본)을 "분석"하고, 당신이 "분석을 편집"하여,
Ept가 분석에 따라 추천한 "슬라이드를 선택"한다면, "발표자료 획득"이 가능한 구조입니다.

하지만, 분석기의 성능이 계속해서 나아지고 있는 중이기 때문에
입력하시는 글은 아래의 Tip을 따르신다면 훨씬 나은 결과를 받으실 수 있을 거에요.


  • Tip 1. [ 낱말 수정/삭제 기능 ]

    [ 낱말을 누르시면, 수정/삭제가 가능해요! ]

  • Tip 2. [ 낱말 합치기 기능 ]

    [ 낱말을 클릭, 드래그하시면, 선택한 낱말 + " " + 도착 낱말로 합쳐집니다. ]

  • Tip 3. [ 문장 추가/삭제 기능 ]

    [ 슬라이드에 해당하는 문장을 추가 또는 삭제할 수 있습니다. ]

  • Tip 4. [ 낱말 추가 기능 ]

    [ 낱말 역시 마음껏 추가가 가능해요! ]
